<Note>
  This guide sets up Home Assistant in a Docker container from scratch. The same steps apply if you already have a running instance or used a different [installation method](https://www.home-assistant.io/installation/).
</Note>

## What you'll need

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) and [Docker Compose](https://docs.docker.com/compose/install/) installed.
- [ngrok](https://download.ngrok.com) installed.

## 1. Initial setup

Create a directory called `home-assistant` containing a `compose.yaml` file and a `config` subdirectory.

## 2. Run Home Assistant in a Docker container

Add this to `compose.yaml`:

```yaml
services:
  homeassistant:
    image: "ghcr.io/home-assistant/home-assistant:stable"
    volumes:
      - /etc/localtime:/etc/localtime:ro
      - /run/dbus:/run/dbus:ro
      - /Desktop/home-assistant/config:/config
    restart: unless-stopped
    ports:
      - 8123:8123
    privileged: true
```

Start the container:

```bash
docker compose up -d
```

Visit `localhost:8123` to confirm the Home Assistant login page loads.

## 3. Configure ngrok to expose Home Assistant

Add the ngrok service to `compose.yaml`:

```yaml
services:
  ngrok:
    image: ngrok/ngrok:latest
    command:
      - "http"
      - "http://host.docker.internal:8123"
    ports:
      - 4041:4041
    environment:
      # find this at https://dashboard.ngrok.com/get-started/your-authtoken
      NGROK_AUTHTOKEN: YOUR_AUTH_TOKEN
```

Restart:

```bash
docker compose up -d
```

Visit your [ngrok dashboard](https://dashboard.ngrok.com/endpoints) to find your endpoint URL. Visiting it now returns a 4xx error—the Home Assistant configuration needs to trust the reverse proxy first.

## 4. Allow trusted proxies in Home Assistant

Open `/home-assistant/config/configuration.yaml` and append:

```yaml
http:
  use_x_forwarded_for: true
  trusted_proxies:
    # use the IP from the error message in your logs if different
    - 192.168.65.1
    - 127.0.0.1
    - ::1
```

Restart the container, then visit your ngrok endpoint URL to access Home Assistant.
