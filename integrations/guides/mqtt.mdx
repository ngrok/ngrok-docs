title: MQTT
description: Learn how to expose your MQTT server to the internet using ngrok TCP tunnels and connect remotely with MQTT Explorer.
---

## What you'll need

- An [ngrok account](https://ngrok.com/signup).
- The [ngrok agent](https://download.ngrok.com) installed.
- [MQTT Explorer](https://mqtt-explorer.com/) installed.
- The [Mosquitto binary](https://mosquitto.org/download/) installed.

## 1. Start an MQTT server

Start Mosquitto (runs on port `1883` by default):

```bash
mosquitto
```

## 2. Launch ngrok

Run:

```bash
ngrok tcp 1883
```

Copy the TCP address ngrok displays—you'll need it for MQTT Explorer.

## 3. Connect with MQTT Explorer

Open MQTT Explorer, click the plus sign to add a new connection, enter the TCP address from step 2, and click **Connect**.

To test the connection, subscribe to a topic in a new terminal tab:

```bash
mosquitto_sub -t 'example/topic' -v
```

Then publish a message to `example/topic` from MQTT Explorer. The message appears in the terminal:

```bash
example/topic hello world
```
