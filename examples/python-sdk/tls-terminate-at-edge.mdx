```python
import ngrok

listener = ngrok.connect("localhost:8080", authtoken_from_env=True,
    proto="tls"
    crt=bytearray(),
    key=bytearray())

print(f"Ingress established at: {listener.url()}");
```

An empty certificate and key will default to the ngrok edge's automatically provisioned keypair.
