## Behavior

### Event logs

When this action is executed, the specified metadata will be collected on the
[tcp_connection_closed.v0](/docs/obs/reference/#tcp_connection_closedv0) event
under the `traffic_policy.logs` key.

### CEL Interpolation

You can access Traffic Policy variables and embed CEL expressions in this
actions configuration values using CEL Interpolation in the form of
`${expression}`. Check out [the examples below](#examples) to see how
CEL Interpolation works.

For a complete list of variables and how to write expressions, see
[Expressions](/tls/traffic-policy/expressions/variables/) and
[Variables](/tls/traffic-policy/expressions/) documentation.
