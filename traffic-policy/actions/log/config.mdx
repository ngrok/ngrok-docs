## Configuration Reference

This is the [Traffic Policy](/docs/http/traffic-policy/) configuration
reference for this action.

### Action Type

`log`

### Configuration Fields

| Parameter  | Type             | Description                                                                                                                                                |
| ---------- | ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `metadata` | `map[string]any` | **Required.** A key-value map of metadata that you would like to include in your events for this action. Supports [CEL Interpolation](#cel-interpolation). |

### Supported Phases

- `on_http_request`
- `on_http_response`
- `on_tcp_connect`
