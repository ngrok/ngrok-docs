## Configuration Reference

This is the [Traffic Policy](/docs/http/traffic-policy/) configuration
reference for this action.

### Action Type

`verify-webhook`

### Configuration Fields

| Parameter  | Type     | Description                                                                                                           |
| ---------- | -------- | --------------------------------------------------------------------------------------------------------------------- |
| `provider` | `string` | **Required.** The name of the provider to verify webhook requests from. Value must be one of the supported providers. |
| `secret`   | `string` | A secret key used to validate requests from the given provider.                                                       |
| `enforce`  | `bool`   | Default `true`. If `false`, continue to the next action even if the request is not allowed.                                                          |

### Supported Directions

- `on-http-request`

### Supported Schemes

- `https`
- `http`
