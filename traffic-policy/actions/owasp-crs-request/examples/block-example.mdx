### Running in block mode

The following configuration demonstrates how to run the `owasp-crs-request` action in block mode.

#### Example Traffic Policy Document

<ConfigExample
	snippetText={null}
	showLineNumbers={true}
	yamlMetastring="{4-11}"
	jsonMetastring="{5-11}"
	config={{
		on_http_request: [
			{
				actions: [
					{
						type: "owasp-crs-request",
						config: {
              on_error: "halt",
						},
          },
				],
			},
		],
	}}
/>
