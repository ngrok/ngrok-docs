## Message
Wildcard names must be reserved exactly.
Failed to create a wildcard endpoint with the domain `DOMAIN` for the account `ACCOUNT`.
This name matches `MATCH` which is reserved for your account, but
wildcard domains must be reserved with an exact match.

Reserve this name on your dashboard: https://dashboard.ngrok.com/domains/new
