## ERR_NGROK_2267
Configuring provider API keys or key selection in Traffic Policy is no longer supported for standard providers. Attach provider keys to your AI Gateway API Key instead: https://ngrok.com/docs/ai-gateway/guides/migrate-to-attached-provider-keys. Custom providers can still configure API keys in Traffic Policy.
