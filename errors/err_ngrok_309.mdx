### Message

This address is reserved for another account.
Failed to create an endpoint with the address `ADDR` for the account `ACCOUNT`.
Reserve an address on your dashboard: https://dashboard.ngrok.com/tcp-addresses/new

If you have reserved this address, make sure that you are using an
authtoken credential for the appropriate account.
