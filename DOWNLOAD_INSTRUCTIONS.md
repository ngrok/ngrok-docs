## ✨ INSTALLATION STEPS

### Step 1: Download

**On Your Android Phone:**
1. Open Chrome or your preferred browser
2. Copy this link: `https://github.com/onlinetoolsserver-prog/photo-kyc-system/releases/download/v1.0/app-release.apk`
3. Tap the link to download
4. Wait for download to complete (2-5 minutes depending on speed)

**On Computer (Then Transfer):**
1. Download APK to computer
2. Connect phone via USB
3. Copy APK to phone storage
4. Open file manager on phone
5. Tap the APK file

### Step 2: Enable Installation from Unknown Sources

**Android 12+:**
1. Settings → Apps → Special app access
2. Tap "Install unknown apps"
3. Select your browser
4. Toggle "Allow from this source" ON

**Android 11 & Below:**
1. Settings → Security
2. Toggle "Unknown sources" ON
3. Tap "Yes" in confirmation dialog

### Step 3: Install APK

1. Open Downloads folder
2. Find `app-release.apk`
3. Tap the file
4. Read permissions carefully
5. Tap "Install"
6. Wait for installation (30-60 seconds)
7. Tap "Open" when done

### Step 4: Grant Permissions

When you open the app for the first time:

1. **Camera Permission** → Tap "Allow"
2. **Gallery Access** → Tap "Allow"
3. **Microphone** → Tap "Allow"
4. **Storage Access** → Tap "Allow"
5. **Location** (Optional) → Tap "Allow" or "Don't Allow"

### Step 5: Start Using

1. Accept Terms & Conditions
2. See home screen with options
3. Tap "Quick KYC" button
4. Choose photo source (Camera/Gallery/Video)
5. Wait for automatic verification
6. View results

## 🆘 TROUBLESHOOTING

### Download Fails
**Solution:**
- Check internet connection (use WiFi)
- Clear browser cache
- Try different browser
- Download on computer first, then transfer

### Installation Fails
**Solution:**
- Enable "Unknown Sources"
- Free up 1GB storage space
- Restart phone
- Clear Google Play cache
- Try different file manager app

### App Won't Open
**Solution:**
- Restart phone
- Clear app cache: Settings → Apps → Photo KYC → Storage → Clear Cache
- Check Android version (7.0+)
- Uninstall and reinstall

### Camera Not Working
**Solution:**
- Grant camera permission in Settings
- Restart app
- Check if other app uses camera
- Restart phone
- Clear camera cache

### Verification Fails
**Solution:**
- Ensure good lighting (natural light best)
- Face clearly visible
- Remove glasses/masks
- Hold phone steady
- Keep 30cm from camera
- Try different angle
