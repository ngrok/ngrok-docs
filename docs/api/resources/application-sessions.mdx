# Application Sessions
------------------

## Get Application Session

Get an application session by ID.

### Request

GET /app/sessions/{id}

<ApplicationSessionsGetRequest />

### Response

Returns a 200 response on success

<ApplicationSessionsGetResponse />

#### Fields

| &nbsp; | &nbsp; | &nbsp; |
|---|---|---|
| `id` | string | unique application session resource identifier |
| `uri` | string | URI of the application session API resource |
| `public_url` | string | URL of the hostport served by this endpoint |
| `browser_session` | [BrowserSession](#api-application-sessions-get-fields-browser-session) | browser session details of the application session |
| `application_user` | [Ref](#api-application-sessions-get-fields-ref) | application user this session is associated with |
| `created_at` | string | timestamp when the user was created in RFC 3339 format |
| `last_active` | string | timestamp when the user was last active in RFC 3339 format |
| `expires_at` | string | timestamp when session expires in RFC 3339 format |
| `endpoint` | [Ref](#api-application-sessions-get-fields-ref) | ephemeral endpoint this session is associated with |
| `edge` | [Ref](#api-application-sessions-get-fields-ref) | edge this session is associated with, null if the endpoint is agent-initiated |
| `route` | [Ref](#api-application-sessions-get-fields-ref) | route this session is associated with, null if the endpoint is agent-initiated |

#### BrowserSession fields

| &nbsp; | &nbsp; | &nbsp; |
|---|---|---|
| `user_agent` | [UserAgent](#api-application-sessions-get-fields-user-agent) | HTTP User-Agent data |
| `ip_address` | string | IP address |
| `location` | [Location](#api-application-sessions-get-fields-location) | IP geolocation data |
#### UserAgent fields

| &nbsp; | &nbsp; | &nbsp; |
|---|---|---|
| `raw` | string | raw User-Agent request header |
| `browser_name` | string | browser name (e.g. Chrome) |
| `browser_version` | string | browser version (e.g. 102) |
| `device_type` | string | type of device (e.g. Desktop) |
| `os_name` | string | operating system name (e.g. MacOS) |
| `os_version` | string | operating system version (e.g. 10.15.7) |
#### Location fields

| &nbsp; | &nbsp; | &nbsp; |
|---|---|---|
| `country_code` | string | ISO country code |
| `latitude` | float64 | geographical latitude |
| `longitude` | float64 | geographical longitude |
| `lat_long_radius_km` | uint64 | accuracy radius of the geographical coordinates |
#### Ref fields

| &nbsp; | &nbsp; | &nbsp; |
|---|---|---|
| `id` | string | a resource identifier |
| `uri` | string | a uri for locating a resource |

## Delete Application Session

Delete an application session by ID.

### Request

DELETE /app/sessions/{id}

<ApplicationSessionsDeleteRequest />

### Response

Returns a 204 response with no body on success
