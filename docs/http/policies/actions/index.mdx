# Actions

Use policy actions to modify the behavior of traffic flowing through your endpoints.

| Type                                               | Description                                                                                                    |
| -------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| [custom-response](custom-response)                 | Send back a pre-defined custom response to the client.                                                         |
| [deny](deny)                                       | Reject incoming traffic to an endpoint.                                                                        |
| [log](log)                                         | Emit an event with custom defined metadata.                                                                    |
| [rate-limit](rate-limit)                           | Rate limit traffic to your upstream servers.                                                                   |
| [url-rewrite](url-rewrite)                         | Transform request URLs using regex.                                                                            |
