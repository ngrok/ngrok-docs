# ERR_NGROK_9009

### Message

Running the agent with an http/s proxy is an enterprise feature. If this is unexpected, verify that there is no proxy_url value in your ngrok configuration file and that the http_proxy environment variable is not set.

If you need this capability, upgrade to an Enterprise plan at: https://dashboard.ngrok.com/billing/subscription

### Further help

If you're having trouble resolving this error, please reach out to [support@ngrok.com](mailto:support@ngrok.com?subject=Help%20with%20ERR_NGROK_9009)
