# ERR_NGROK_1119

### Message
Cannot use binding address '-R `ADDR`:' with '-subdomain=`SUBDOMAIN` parameter.
You may either specify binding address '-R `ADDR`:' or '-subdomain=`SUBDOMAIN` but not both.
