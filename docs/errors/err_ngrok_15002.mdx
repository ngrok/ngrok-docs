# ERR_NGROK_15002

### Message

Your account is on the `PLAN` plan, which requires you to register a Domain through the ngrok Dashboard before starting an Endpoint. Visit https://dashboard.ngrok.com/cloud-edge/domains to register a domain and then use the `--domain` flag when starting your Agent.

### Further help

If you're having trouble resolving this error, please reach out to [support@ngrok.com](mailto:support@ngrok.com?subject=Help%20with%20ERR_NGROK_15002)
