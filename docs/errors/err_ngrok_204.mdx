# ERR_NGROK_204

### Message
Only the Personal, Pro, and Enterprise accounts can use the ngrok API.
This is a valid API token, but the account `ACCOUNT` is not authorized to use the ngrok API. 

Upgrade to a paid plan at: https://dashboard.ngrok.com/billing/subscription
