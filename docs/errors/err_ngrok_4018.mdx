# ERR_NGROK_4018

### Message

You must have a valid account and authtoken to perform this operation.

If you are seeing this error, make sure you have done the following:

- Sign up for an account: https://dashboard.ngrok.com/signup
- If you have an account, install your authoken: https://dashboard.ngrok.com/get-started/your-authtoken

### Further help

If you're having trouble resolving this error, please reach out to [support@ngrok.com](mailto:support@ngrok.com?subject=Help%20with%20ERR_NGROK_4018)
