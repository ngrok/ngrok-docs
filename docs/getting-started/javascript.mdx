ngrok-javascript is the official Node.js SDK for ngrok that requires no binaries. Quickly enable secure production-ready connectivity to your applications and services directly from your code.

With ngrok-javascript, you can serve Node.js applications on the internet in a single line of code without configuring low-level network primitives like IPs, certificates, load balancers, and ports.
