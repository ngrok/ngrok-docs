## Time variables

The following variables are available under the `time` namespace:

| Name                   | Type   | Description                             |
| ---------------------- | ------ | --------------------------------------- |
| [`time.now`](#timenow) | string | The current UTC time in RFC3339 format. |

### `time.now`

The current UTC time in RFC3339 format.
