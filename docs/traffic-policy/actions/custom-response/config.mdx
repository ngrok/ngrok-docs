## Configuration Reference

This is the [Traffic Policy](/docs/http/traffic-policy/index.mdx) configuration 
reference for this action.

### Action Type

`custom-response`

### Configuration Fields

| Parameter     | Type                    | Description                                                                                                                                        |
| ------------- | ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| `status_code` | `int`                     | **Required.** Status code of the custom response being sent back.                                                                                                                 |
| `content`     | `string`                  | Body of the custom response being sent back. Supports [string interpolation](#string-interpolation).                                             |
| `headers`     | `map[string]string`       | Map of key-value headers of the custom response to be sent back. If `content-type` is not included in `headers`, this action will attempt to infer the correct `content-type`. |

### Supported Directions

- `inbound`
- `outbound`

### Supported Schemes

- `https`
- `http`
