## Configuration Reference

This is the [Traffic Policy](/docs/http/traffic-policy/index.mdx) configuration 
reference for this action.

### Action Type

`url-rewrite`

### Configuration Fields

| Parameter     | Type                      | Description                                                                                                                                                           |
| ------------- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `from`        | `string`                    | **Required.** A regular expression string to match inside of the URL. Supports [string interpolation](#string-interpolation).                                                     |
| `to`          | `string`                    | **Required.** A regular expression string to replace the `from` match with. Supports [string interpolation](#string-interpolation).                                 |
| `format`      | `string`                    | Determines interpolation format in `to`/`from` fields. Default `ngrok`.                                                                                             |

### Supported Directions

- `inbound`

### Supported Schemes

- `https`
- `http`
