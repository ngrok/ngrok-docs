# Overview

The Traffic Policy module enables you to filter inbound and outbound traffic with [Common Expression Language (CEL)](https://github.com/google/cel-spec) expressions. Each policy rule expression must evaluate to true in order for
the rule's actions to take effect against traffic.

- [Learn how to write CEL expressions](/traffic-policy/concepts/expressions/#writing-expressions).

In addition to CEL's [built-in functions and macros](https://github.com/google/cel-spec/blob/master/doc/langdef.md#macros) we provide additional variables and macros:

- [Action Result Variables](/docs/traffic-policy/variables/connection#action-result-variables)
- [Connection Variables](/docs/traffic-policy/variables/connection#connection-variables)
- [Connection Geo Variables](/docs/traffic-policy/variables/connection#connection-geo-variables)
- [Endpoint Variables](/docs/traffic-policy/variables/connection#endpoint-variables)
- [Time Variables](/docs/traffic-policy/variables/connection#time-variables)
- [List of available macros for TCP](/traffic-policy/macros)
