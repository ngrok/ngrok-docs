title: What's new?

At ngrok, we deploy our SaaS platform multiple times a day across our global network. We are constantly improving and adding new features.

You can expect this page to update regularly (at least monthly). We'll include things like:

- Feature Launches with links to blogs and other release material
- Major and minor agent, SDK, and API client releases
- ngrok Dashboard improvements
- Changes to our API
- Major security fixes
