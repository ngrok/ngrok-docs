# Site-to-Site Connectivity

ngrok allows you to connect to remote sites without asking your customers and partners to make network changes.

This section provides getting started guides for connecting to remote sites with ngrok.

| Name                                                                   | Description                                                                                                                                     |
| ---------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| [APIs](apis.mdx)                                                       | Connect to APIs at remote sites                                                                                                                 |
| [APIs with mTLS](apis-mtls.mdx)                                        | Connect to APIs at remote sites using end-to-end encryption with mTLS                                                                           |
| [DBs](dbs.mdx)                                                         | Connect to databases at remote sites                                                                                                            |
| [DBs with mTLS](dbs-mtls.mdx)                                          | Connect to databases at remote sites using end-to-end encryption with mTLS and tunnels                                                          |
| [Site-to-site installation guide for end customers](end-customers.mdx) | Get details on why your vendor has requested you install ngrok for site-to-site connectivity, how it works, and how to maintain the ngrok agent |
