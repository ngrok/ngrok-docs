[DeepSeek](https://www.deepseek.com) provides high-performance reasoning and chat models. DeepSeek requires you to bring your own key—ngrok-managed keys are not available for DeepSeek.

## Setup

<Steps>
  <Step title="Create an AI Gateway endpoint">
    If you don't have one yet, follow the [quickstart](/ai-gateway/quickstart) to create your AI Gateway endpoint.
  </Step>

  <Step title="Get a DeepSeek API key">
    Sign up at [platform.deepseek.com](https://platform.deepseek.com) and create an API key.
  </Step>

  <Step title="Make a request">
    Pass your key directly—the gateway forwards it to DeepSeek.

    <CodeGroup>
    ```python Python
    from openai import OpenAI

    client = OpenAI(
        base_url="https://your-ai-gateway.ngrok.app/v1",
        api_key="sk-..."  # Your DeepSeek key, forwarded by gateway
    )

    response = client.chat.completions.create(
        model="deepseek-chat",
        messages=[{"role": "user", "content": "Hello!"}]
    )

    print(response.choices[0].message.content)
    ```

    ```typescript TypeScript
    import OpenAI from "openai";

    const client = new OpenAI({
      baseURL: "https://your-ai-gateway.ngrok.app/v1",
      apiKey: "sk-...",  // Your DeepSeek key, forwarded by gateway
    });

    const response = await client.chat.completions.create({
      model: "deepseek-chat",
      messages: [{ role: "user", content: "Hello!" }],
    });

    console.log(response.choices[0].message.content);
    ```

    ```typescript Vercel AI SDK
    import { generateText } from "ai";
    import { createOpenAI } from "@ai-sdk/openai";

    const gateway = createOpenAI({
      baseURL: "https://your-ai-gateway.ngrok.app/v1",
      apiKey: "sk-...",  // Your DeepSeek key, forwarded by gateway
    });

    const { text } = await generateText({
      model: gateway("deepseek-chat"),
      prompt: "Hello!",
    });

    console.log(text);
    ```

    ```bash cURL
    curl https://your-ai-gateway.ngrok.app/v1/chat/completions \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer sk-..." \
      -d '{
        "model": "deepseek-chat",
        "messages": [{"role": "user", "content": "Hello!"}]
      }'
    ```
    </CodeGroup>
  </Step>
</Steps>

## Store key in the gateway

Instead of each client passing their own key, you can store it once in ngrok Secrets and have the gateway inject it automatically.

<Warning>
Storing your provider key in the gateway makes your endpoint publicly accessible. You must add authorization to prevent unauthorized use and unexpected charges. See [Protecting BYOK Endpoints](/ai-gateway/guides/protecting-byok-endpoints).
</Warning>

<Steps>
  <Step title="Store your key in ngrok secrets">
    ```bash
    ngrok api secrets create \
      --name deepseek \
      --secret-data '{"api-key": "sk-..."}'
    ```

    Or use the [Vaults & Secrets dashboard](https://dashboard.ngrok.com/vaults).
  </Step>

  <Step title="Configure your traffic policy">
    ```yaml
    on_http_request:
      - type: ai-gateway
        config:
          providers:
            - id: "deepseek"
              api_keys:
                - value: ${secrets.get('deepseek', 'api-key')}
    ```
  </Step>

  <Step title="Make a request">
    Clients no longer need a DeepSeek key—pass any value for `api_key`.

    <CodeGroup>
    ```python Python
    from openai import OpenAI

    client = OpenAI(
        base_url="https://your-ai-gateway.ngrok.app/v1",
        api_key="unused"  # Gateway injects your DeepSeek key
    )

    response = client.chat.completions.create(
        model="deepseek-chat",
        messages=[{"role": "user", "content": "Hello!"}]
    )

    print(response.choices[0].message.content)
    ```

    ```typescript TypeScript
    import OpenAI from "openai";

    const client = new OpenAI({
      baseURL: "https://your-ai-gateway.ngrok.app/v1",
      apiKey: "unused",  // Gateway injects your DeepSeek key
    });

    const response = await client.chat.completions.create({
      model: "deepseek-chat",
      messages: [{ role: "user", content: "Hello!" }],
    });

    console.log(response.choices[0].message.content);
    ```

    ```typescript Vercel AI SDK
    import { generateText } from "ai";
    import { createOpenAI } from "@ai-sdk/openai";

    const gateway = createOpenAI({
      baseURL: "https://your-ai-gateway.ngrok.app/v1",
      apiKey: "unused",  // Gateway injects your DeepSeek key
    });

    const { text } = await generateText({
      model: gateway("deepseek-chat"),
      prompt: "Hello!",
    });

    console.log(text);
    ```

    ```bash cURL
    curl https://your-ai-gateway.ngrok.app/v1/chat/completions \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer unused" \
      -d '{
        "model": "deepseek-chat",
        "messages": [{"role": "user", "content": "Hello!"}]
      }'
    ```
    </CodeGroup>
  </Step>
</Steps>
