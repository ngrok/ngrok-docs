---
title: Credits
description: Understanding AI Gateway credits, pricing, and balance management.
---

AI Gateway credits are a prepaid balance that funds all AI Gateway usage. Every request through the AI Gateway — whether using [AI Gateway API Keys](/ai-gateway/concepts/api-keys) or [your own provider keys](/ai-gateway/bring-your-own-keys/overview) — consumes credits.

## Plan requirement

AI Gateway requires the **Pay-as-you-go** plan. Free and Hobbyist plans cannot use the AI Gateway.

## Purchasing credits

Purchase credits through the [ngrok dashboard](https://dashboard.ngrok.com/ai-gateways/credit/checkout):

1. Navigate to **AI Gateways** in the dashboard
2. Click **Buy Credits**
3. Enter the amount (minimum **$5.00**)
4. Complete checkout

Credits expire **365 days** after purchase. There is no REST API for purchasing credits — this is a dashboard-only operation.

## How credits are consumed

Credits cover two fee components:

| Fee | Applies to | Cost |
|-----|-----------|------|
| **Processing fee** | All requests (AI Gateway API Keys and BYOK) | $0.05 per million tokens |
| **Upstream provider fee** | AI Gateway API Keys only | Varies by provider and model |

- **Processing fee**: ngrok's fee for routing and processing every request through the AI Gateway. This applies to all requests regardless of authentication method.
- **Upstream provider fee**: When you use [AI Gateway API Keys](/ai-gateway/concepts/api-keys), ngrok pays the upstream provider (OpenAI, Anthropic) on your behalf. This cost is also deducted from your credits.
- **BYOK**: Only the processing fee is charged via credits. You pay the provider directly for AI model costs through your own billing relationship.

## Balance tracking

View your current credit balance in the [ngrok dashboard](https://dashboard.ngrok.com/ai-gateways). The balance display shows:

- **Current balance**: Remaining credit amount
- **Expiration date**: When each credit block expires

## Low balance alerts

When your balance drops below a threshold, ngrok sends an email notification warning you to purchase additional credits before they run out.

## What happens when credits run out

When your credit balance reaches zero:

1. **All AI Gateway usage stops** — both AI Gateway API Key requests and BYOK requests are blocked
2. Clients receive **HTTP 403** with error [`ERR_NGROK_4026`](/errors/err_ngrok_4026): "Your account has run out of credit"
3. The `addon_ai_gateway_ngrok_managed_tokens` addon is revoked from your account

<Warning>
Credit depletion affects **all** AI Gateway usage, including BYOK. Even if you manage your own provider keys, you still need credits to cover the processing fee.
</Warning>

## Restoring access

Purchase additional credits through the dashboard. Once your balance is restored:

- The addon is automatically re-enabled
- AI Gateway API Keys resume working immediately
- BYOK requests resume working immediately

## AI Gateway credits vs. platform credits

AI Gateway credits are separate from general ngrok platform credits (used for endpoints, bandwidth, etc. on Free/Hobbyist plans). They are:

- Purchased separately
- Tracked separately
- Used exclusively for AI Gateway requests
