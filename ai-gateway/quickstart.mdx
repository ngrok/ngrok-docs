Complete the ngrok AI Gateway SDK integration (I've already signed up on app.ngrok.ai, purchased credits, and have an access key):

1. Ask me for my access key (format: ng-xxxxx-g1-xxxxx). The gateway URL is always https://gateway.ngrok.ai
2. Install the OpenAI SDK: `npm install openai` or `pip install openai`
3. Write a script that routes OpenAI SDK requests through the gateway by setting baseURL to https://gateway.ngrok.ai/v1 and apiKey to my ng-xxxxx key
4. Send a test request with model `gpt-4o` and message "State your exact model name and provider."
5. Print and verify the response

Ask me for my access key before writing any code.
</Prompt>

<Note>
This quickstart uses ngrok.ai inference for built-in OpenAI and Anthropic providers. Every request consumes [credits](/ai-gateway/concepts/credits)—credits cover both the processing fee and upstream model cost. To bring your own key instead, see [Provider Keys](/ai-gateway/guides/attaching-provider-keys).
</Note>

## Before you start

Sign in at [app.ngrok.ai](https://app.ngrok.ai). You don't need a specific ngrok subscription plan—purchase AI Gateway credits to get started.

<Steps>
  <Step title="Sign in to app.ngrok.ai">
    Open [app.ngrok.ai](https://app.ngrok.ai) and sign in with your ngrok account, then complete onboarding if prompted.
  </Step>

  <Step title="Add credits">
    Open the [Credits](https://app.ngrok.ai/credits) page and add at least $5.00. Every request draws down credits for the AI Gateway's [processing fee](/ai-gateway/concepts/credits), and ngrok.ai inference for OpenAI or Anthropic also bills upstream model cost to your credits.
  </Step>

  <Step title="Create an access key">
    Open [Access Keys](https://app.ngrok.ai/keys) and create a new key. Copy the token right away—it's shown only once—and store it somewhere safe. It looks like `ng-xxxxx-g1-xxxxx`.
  </Step>

  <Step title="Make your first request">
    Point your OpenAI SDK at the gateway using your access key:

    ```js
    import OpenAI from "openai";

    const openai = new OpenAI({
      baseURL: "https://gateway.ngrok.ai/v1",
      apiKey: "ng-xxxxx-g1-xxxxx"
    });

    const res = await openai.chat.completions.create({
      model: "gpt-4o",
      messages: [{ role: "user", content: "Hello, world!" }]
    });
    ```

    To confirm requests route through the gateway, ask the model to `State your exact model name and provider.` and check the response.
  </Step>
</Steps>
