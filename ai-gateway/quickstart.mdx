## Before you start

You'll need:
- An [ngrok account](https://dashboard.ngrok.com/signup) on the **Pay-as-you-go** plan
- No AI provider accounts needed—ngrok manages the provider keys for you

<Steps>
  <Step title="Login to the ngrok Dashboard">
    Go to [dashboard.ngrok.com](https://dashboard.ngrok.com) and sign in to your account.
  </Step>

  <Step title="Navigate to AI Gateways">
    In the left sidebar, click [**AI Gateways**](https://dashboard.ngrok.com/ai-gateways) under the Universal Gateway section.
  </Step>

  <Step title="Create a New AI Gateway">
    Click the **+ New AI Gateway** button. Enter a URL for your endpoint (for example, `https://your-ai-gateway.ngrok.app`). If the domain is available, it will automatically be reserved for you. Click **Create & Configure** to finish setup.
  </Step>

  <Step title="Purchase Credits">
    AI Gateway usage requires [prepaid credits](/ai-gateway/concepts/credits). [Purchase at least $5.00 in credits](https://dashboard.ngrok.com/ai-gateways/credit/add) to get started. Credits fund both the processing fee and the upstream provider costs.
  </Step>

  <Step title="Create an AI Gateway API Key">
    In your gateway's detail page, go to the **API Keys** tab and click **New API Key**. Copy the key immediately—it's only shown once.

    You can also create keys via the CLI:

    ```bash
    ngrok api ai-gateway-api-keys create \
      --endpoint-id ep_xxxxx \
      --description "My app"
    ```
  </Step>

  <Step title="Integrate with Your Application">
    Update your SDK configuration to point to your AI Gateway endpoint, using the AI Gateway API Key you just created:

    ```js
    import OpenAI from "openai";

    const openai = new OpenAI({
      baseURL: "https://your-ai-gateway.ngrok.app/v1",
      apiKey: "ng-xxxxx-g1-xxxxx"  // Your AI Gateway API Key
    });
    ```

    Then use your SDK as normal:

    ```js
    const res = await openai.chat.completions.create({
      model: "gpt-4o",
      messages: [{ role: "user", content: "Hello, world!" }]
    });
    ```
  </Step>
</Steps>

<Warning>
Your AI Gateway API Key token is only shown once at creation time. Store it securely. If you lose it, delete the key and create a new one.
</Warning>
