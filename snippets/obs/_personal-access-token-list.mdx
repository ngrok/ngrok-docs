| Name | Type | Description |
|---|---|---|
| tokens.id | string | unique identifier of the Personal Access Token |
| tokens.uri | string | URI of this Personal Access Token API resource |
| tokens.description | string | human-readable description of this token |
| tokens.token | string | opaque bearer token secret. **Only present in the Create response.** |
| tokens.created_at | string | timestamp when the token was created, RFC 3339 format |
| tokens.expires_at | string | optional expiration time, RFC 3339 format. Omitted if the token does not expire. |
| tokens.revoked_at | string | optional timestamp when the token was revoked, RFC 3339 format. Omitted if not revoked. |
| uri | string | URI of the Personal Access Token API resource list |
| next_page_uri | string | URI for the next page of results, or null if there are no more pages. |
