| Name | Type | Description |
|---|---|---|
| id | string | unique identifier of the Personal Access Token |
| uri | string | URI of this Personal Access Token API resource |
| description | string | human-readable description of this token |
| token | string | opaque bearer token secret. **Only present in the Create response.** |
| created_at | string | timestamp when the token was created, RFC 3339 format |
| expires_at | string | optional expiration time, RFC 3339 format. Omitted if the token does not expire. |
| revoked_at | string | optional timestamp when the token was revoked, RFC 3339 format. Omitted if not revoked. |
