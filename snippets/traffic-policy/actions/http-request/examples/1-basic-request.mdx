### Make a GET request

Performs a simple GET request to an internal endpoint.

<CodeGroup>
```yaml policy.yml
on_http_request:
  - name: BasicInternalRequest
    actions:
      - type: http-request
        config:
          url: https://upstream-service.internal/ping
```

```json policy.json
{
	"on_http_request": [
		{
			"name": "BasicInternalRequest",
			"actions": [
				{
					"type": "http-request",
					"config": {
						"url": "https://upstream-service.internal/ping"
					}
				}
			]
		}
	]
}
```

</CodeGroup>
