### Basic Example

This example configuration sets up an endpoint (`hotdog.ngrok.app`) and closes connection to the endpoint.

#### Example Traffic Policy Document

```on_http_request:
- name: DC
  expressions:
  - req.url.path.startsWith("/dc")
  actions:
  - type: close-connection
```

#### Start Endpoint with Traffic Policy

```bash
ngrok http 8080 --url hotdog.ngrok.app --traffic-policy-file /path/to/policy.yml
```

#### Helper script

```bash
curl http://hotdog.ngrok.app/dc
curl: (16) Error in the HTTP2 framing layer
```
