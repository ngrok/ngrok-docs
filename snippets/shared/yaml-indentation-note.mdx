<Note>

YAML is whitespace-sensitive.
Use spaces only (tabs are not valid in YAML) and keep keys at the same level aligned to the same column.

✅ Valid:

```yaml
endpoints:
  - name: example
    upstream:
      url: 80
```

❌ Invalid—`upstream` is indented inconsistently and will fail to parse:

```yaml
endpoints:
  - name: example
     upstream:
      url: 80
```

</Note>
